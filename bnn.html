Binarized Neural Network for Digit Recognition on FPGA

  • My lab partner Vidya and I made an Binarized Neural Network for Digit Recognition on FPGA for final project of ECE 5760 “Advanced Microcontroller Design and System On Chip” in Spring 2018.
  • A Binarized Neural Network is a special type of Convolutional Neural Network, which is a machine learning model based on the neural networks of the human brain and has extensive uses in image classification. We coded the Binarized Neural Network in System Verilog on Intel DE1-SoC board and the Binarized Neural Network consists of two convolutional layers, two pooling layers, and two fully connected layers.
  • When inputted with a 7 by 7 two bit black and white image, the Binarized Neural Network on FPGA can make inference and classify what number is the input image in 4us, which takes far less time than the same implementation running Python on a PC (>40us or at least 10 times longer). This hardware acceleration is the result of our parallel calculation algorithm in FPGA. Another advantage of running Binarized Neural Network on FPGA hardware is that it guarantees meeting real time deadline, whereas a typical program running in operating system is usually not real time.
